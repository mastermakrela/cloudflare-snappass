How does it work?

When you submit a password, it is encrypted and stored in the Cloudflare Workers KV.
Then you will receive the only decryption key as a link, which you can share with the recipient.
The password will be deleted after the specified time or after the first decryption.

Is it secure?
Cloudflare Workers KV provides:
  • Encryption-at-rest (EAR).
  • Encryption-in-transit (EIT).
